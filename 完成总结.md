## 📊 完整的数据流

```
用户操作
  ↓
选择/拖拽文件
  ↓
FileUpload.vue 创建文件对象
  ↓
发送到后端 API
  ↓
FastAPI 接收并保存
  ↓
返回文件信息（filename, filepath, size）
  ↓
更新文件状态为 success
  ↓
emit('filesUploaded', successFiles)
  ↓
App.vue 接收并添加到 files 数组
  ↓
FileList.vue 通过 watch 监听到变化
  ↓
v-for 循环渲染文件列表
  ↓
✅ 用户看到上传的文件
```

## 🐛 常见问题

### Q1: 文件列表为空？
**A:** 检查以下步骤：
1. 确认后端已启动：访问 http://localhost:8000/health
2. 打开 F12 查看 Console
3. 上传文件并查看日志
4. 确认看到 "上传成功" 日志
5. 确认看到 "收到上传的文件" 日志

### Q2: 看不到文件路径？
**A:** 这是正常的，文件路径是可选显示项。只有当后端返回 filepath 字段时才会显示。

### Q3: 样式不正常？
**A:** 
1. 清除浏览器缓存（Ctrl+Shift+R）
2. 确认 CSS 样式已加载
3. 检查浏览器开发者工具 Elements 标签
