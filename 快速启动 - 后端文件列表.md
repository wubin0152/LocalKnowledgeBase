## 🔍 快速诊断

### 如果看不到文件列表

**检查后端：**
```bash
# 1. 后端是否运行？
curl http://localhost:8000/api/files/list

# 2. uploads 目录有文件吗？
ls uploads/

# 3. API 正常吗？
python test_file_list.py
```

**检查前端：**
1. 按 F12 打开开发者工具
2. 查看 Console 标签
3. 应该看到日志：
   ```
   📡 [App.vue] 正在获取文件列表...
   📋 [App.vue] 获取到文件列表：X 个文件
   ```

### 如果一直显示加载中

**可能原因：**
- 后端未启动
- 网络请求失败
- CORS 问题

**解决方法：**
1. 确认后端在运行：访问 http://localhost:8000/docs
2. 检查 Console 是否有错误
3. 刷新页面重试
