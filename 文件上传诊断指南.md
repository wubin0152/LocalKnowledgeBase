# 文件上传功能诊断指南

## 🔍 问题诊断步骤

### 第一步：检查后端是否启动

```bash
cd KnowledgeBasePython
python main.py
```

**预期输出：**
```
🚀 启动 Knowledge Base 后端服务...
📖 API 文档：http://localhost:8000/docs
📝 ReDoc 文档：http://localhost:8000/redoc
INFO:     Uvicorn running on http://0.0.0.0:8000
```

如果没有看到这些信息，说明后端没有正常启动。

### 第二步：测试后端 API

**方法 1：使用浏览器访问 Swagger 文档**
打开 http://localhost:8000/docs

- 找到 `POST /api/files/upload` 接口
- 点击 "Try it out"
- 选择一个文件
- 点击 "Execute"
- 查看响应

**方法 2：使用测试脚本**
```bash
cd KnowledgeBasePython
python test_upload_api.py
```

**方法 3：使用独立的 HTML 测试页面**
```bash
cd KnowledgeBaseVue
# 在浏览器中打开 test-upload.html
```

### 第三步：检查前端是否正常

1. **启动前端服务**
```bash
cd KnowledgeBaseVue
npm install
npm run dev
```

2. **打开浏览器访问前端地址**（通常是 http://localhost:5173）

3. **打开浏览器开发者工具**（F12）
   - 查看 Console 是否有错误
   - 查看 Network 标签，检查上传请求

4. **测试上传文件**
   - 点击上传区域
   - 选择一个小文件（如 TXT 文件）
   - 观察上传进度

5. **检查文件列表**
   - 上传成功后，文件应该出现在左侧文件列表
   - 应该有文件名、大小、图标等信息

## 🐛 常见问题及解决方案

### 问题 1：前端看不到上传后的文件列表

**可能原因：**
1. 后端未启动
2. CORS 配置问题
3. 前端事件处理有问题

**解决方案：**

#### 方案 1：确认后端已启动
```bash
cd KnowledgeBasePython
python main.py
```

#### 方案 2：检查浏览器 Console
打开 F12 开发者工具，查看 Console：
- 如果有 CORS 错误，检查后端 CORS 配置
- 如果有网络错误，确认后端地址正确

#### 方案 3：手动测试 API
在浏览器 Console 中运行：
```javascript
fetch('http://localhost:8000/health')
  .then(r => r.json())
  .then(d => console.log('后端状态:', d))
  .catch(e => console.error('后端连接失败:', e))
```

### 问题 2：上传失败

**检查项：**
1. 后端服务是否运行
2. `uploads/` 目录是否存在
3. 文件是否为空

**解决：**
```bash
# 创建 uploads 目录
cd KnowledgeBasePython
mkdir uploads

# 检查目录权限
ls -la uploads/
```

### 问题 3：前端无法连接后端

**检查：**
1. 后端地址配置是否正确
2. 端口是否被占用
3. 防火墙设置

**修改前端配置：**
编辑 `src/utils/api.js`:
```javascript
export const API_CONFIG = {
  BASE_URL: 'http://localhost:8000',  // 确保地址正确
  TIMEOUT: 30000,
}
```

## ✅ 正确的流程演示

### 1. 启动后端
```bash
cd KnowledgeBasePython
pip install fastapi uvicorn python-multipart aiofiles pydantic
python main.py
```

### 2. 启动前端
```bash
cd KnowledgeBaseVue
npm install
npm run dev
```

### 3. 访问前端
打开浏览器访问 http://localhost:5173

### 4. 上传文件
- 点击左侧上传区域
- 选择一个文件（建议先测试小文件）
- 等待上传完成（看到 ✓ 标记）

### 5. 查看文件列表
上传成功的文件会显示在左侧文件列表中，包含：
- 文件图标
- 文件名
- 文件大小
- 操作按钮（预览、下载、删除）

### 6. 与文件对话
在右侧聊天窗口输入问题，系统会基于上传的文件回答。
